# zack-gaming-2024
As part of our TESDA SIL/OJT, we were required to build a gaming event website. Pre-requisites are to use company's colors and font or nearest alternative using wix.com platform

I, along with 4 other TESDA Scholars collaborated on this project, using the free version of wix.

[ZACK Gaming 2024](https://djordab4zack.wixsite.com/zack-gaming-2024)
